FILE FORMATTING

A file format is a standard way that data is encoded to store in a PC. It determines how bits are utilized to encode data in a digital storage medium. File formats might be either restrictive or free and might be either unpublished or open. Amongst all file formats the most commonly used formats on a daily basis are .pdf, .jpeg, and .png. Below we've made a step-by-step tutorial on how to make these file formats ADA compliant.

ESSENTIAL TOOLS


Adobe Acrobat

This tool can be used to fix ADA related issues and to create ADA compliant .pdf or .jpeg files. It also generates an accessibility report for the file so that the user is aware of the things that needs to be rectified.





Light House

Light House is a validator which is commonly used on Websites. This can also be used to generate accessibility report on a specific website that you prefer. It also comes in as a google chrome plugin which makes the users life so easy.





Wave

WAVE is a suite of evaluation tools that helps authors make their web content more accessible to individuals with disabilities. WAVE can identify many accessibility and Web Content Accessibility Guideline (WCAG) errors, but also facilitates human evaluation of web content.
