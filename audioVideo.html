VIDEO SUBTITLES





How to make a HTML5 compatable video

The main way to make your video HTML 5 compatable is to make sure the file is of the correct type. The three most supported formats are MP4, WebM, and Ogg. Using some kind of video conversion tool you can easily make a video into the desired type. You can also edit how certail aspects of the videos bars and menu look to further increase accessibility.









How to give the video subtitles

In order to give a video subtitles you will need to do some HTML coding. Inbetweenn the opening and closing video tag you need to mae a track tag. This should contain a label, kind, src, and srclang. these all describe the folowing. The label says this is going to be a specific language, the kind is what we are trying to make so for this it should be "subtitles", the src will go to a file you create that is a .vtt file (we will talk about this later), and lastly, srclang says what the language the src file is in.







Setting Up .vtt File

This file has specific details required for your video player to make the captions. To make the file you will need to do the following

WEBVTT [enter] [enter] 1 [enter] 00:00:00.00 --> 00:00:01.00 [enter] (CAPTION HERE) [enter] [enter] 2 [enter] 00:00:01.00 --> . . .

WEBVTT lets the browser know its a vtt file, 1 is the first caption, next is the timestamp for when the caption is to appear, then 2 for the second and so on and so forth.
