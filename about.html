ABOUT US


This site was created by students at Montana State University for a Fall 2020 course in Web Development, CSCI 331.

Our objective was to create a site that is fully ADA compliant and simultaneously serves as an educational guide for other-web developers to learn main topics in web accessibility. While researching and implementing the detailed requirements for web accessibility, we gained valuable skills and knowledge for our future professional and personal web development projects.
