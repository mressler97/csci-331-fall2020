SITE PRESENTATION

Color Use

1.4.1: It's important not to use color as the only visual means of conveying information. Another method of communication must be included such as descriptive text, symbols, or shapes.

Example: If the intention is to indicate a form field is missing, don't simply highlight the box in red. Add text that describes what the error is and what can be done to correct it.

Color Contrast

1.4.3: Text and images of text must have a color contrast ratio of 4.5:1 when compared with the background. A color picker can provide the RGB values for the colors on your site. Use an online color contrast checker to find the contrast ratio once your know the RGB values for the text and background colors.

Using Bootstrap Templates

Bootstrap color themes often do not meet the 4.5:1 color contrast ratio. Test the colors provided using a color contrast checker. To override the original styles.css, create a separate custom.css file. Reference it after the original bootstrap file in the HTML. In custom.css, use the same CSS rules to update the color references.

Example: To change the color for the following Bootstrap CSS rule: .navbar-light .navbar-text a {color: #757575;}, update the rule in custom.css, making sure the selector is as specific as the original: .navbar-light .navbar-text a {color: #000000;}.

Sensory Characteristics

1.3.3: Any instructions provided on a website must not rely solely on sensory characteristics such as shape, color, size, visual location, orientation, or sound. Additional clarifying information is required. Some users, such as a blind user accessing the web using a screen reader, are not able to perceive shape or position (for example, "click the button to the left").

Example: A table displaying a bus schedule uses both color and shape to identify the bus arrival times. The table displays bus stops along the top row and times in the first column. Bus arrival times are marked in the appropriate cell using a diamond shaped glyph with a specific background color so the information is communicated by both color and shape.

Images of Text

1.4.5: Always use text instead of using images of text. Exceptions to this guideline include logos, where it is acceptable to use images of text as well as text that is part of an image containing significant other visual content, such as a graph or diagram. Images of text do not allow adjustment by people who require certain fonts, text sizes, or background and foreground colors.
